# Precedent Study — Part 1

This repository contains a minimal scaffold for the Part 1 Precedent Study datasheet and instructions to publish a simple website for submission.

Publish via GitHub Pages (quick):

1. Create a GitHub repo and push this project.
2. In the repository settings, under "Pages", set the source to the `main` branch and the `/website` folder (or `gh-pages` branch if you prefer).
3. After saving, GitHub will provide a URL for your site. Submit that URL for the assignment.

If you need step-by-step help, follow the Computational Design Workflows tutorials 1 + 2 from the CDW seminar.
